(static) sleep(time) → {Promise}

Description:
  • sleep函数，可在异步场景中实现延时或者暂停执行
Source:
  • utils.js, line 21
Example
async function batchUpdate(datas) {
  // 批量调用接口更新数据，每次更新之间等待1秒钟以防止频繁请求导致接口被限制
  for (const data of datas) {
    await doUpdate(data);
    await sleep(1000);
  }
}
