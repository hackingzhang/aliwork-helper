# static generateDptFieldData(id, name) → {object}

生成部门字段数据，可直接赋值给部门组件
⚠️如果传数组作为参数，ID和名称的顺序必须一一对应。
🤯通过数据源获取到的表单数据，部门字段的值就是部门ID和部门名称分开的两个数组， 可以直接将其传入来生成可赋值给部门组件的部门字段数据。
